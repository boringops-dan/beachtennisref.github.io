# What is the service sequence in doubles beach tennis?

**Short Answer:** In doubles, service rotates among all four players. Team A's first player serves game 1, Team B's first player serves game 2, Team A's second player serves game 3, Team B's second player serves game 4, then the cycle repeats. Order is set at the start of each set.

## Full Explanation

Per ITF Beach Tennis Rules Rule 8, in doubles each player serves for an entire game. At the start of each set, each team decides which partner will serve first that set. The service order then rotates: Team A Player 1 serves game 1, Team B Player 1 serves game 2, Team A Player 2 serves game 3, Team B Player 2 serves game 4, and so on. This four-player rotation continues throughout the set. Similarly, the receiving order is fixed at the start of each set: each receiver alternates games within their team and must receive in the same half of the court (deuce or advantage) throughout the set. Service and receiving orders may be changed at the start of a new set. BeachTennisRef.App tracks and enforces this rotation automatically.

**Ruleset:** ITF
**Rule Reference:** ITF Beach Tennis Rules, Rule 8 (Order of Service in Doubles)
