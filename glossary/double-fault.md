# Double Fault - Beach Tennis Glossary

> A double fault occurs when the server commits two consecutive service faults on the same point, resulting in the point being awarded to the receiver. Per ITF Beach Tennis Rules Rule 7, the server has two attempts to put the ball into play; committing a fault on both attempts loses the point regardless of the score. In no-ad scoring at deuce (40-40), the receiver still wins the golden point if the server double-faults. A let does not count as a fault and resets that particular service attempt.

**Ruleset:** ITF
**Rule Reference:** ITF Beach Tennis Rules, Rule 7 (The Service)

**Related Terms:** service-fault, foot-fault, let, golden-point
