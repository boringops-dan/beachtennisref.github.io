# Golden Point - Beach Tennis Glossary

> The golden point is the decisive single point played when a game reaches deuce (40-40) under no-advantage scoring. In beach tennis, standard scoring uses no-advantage (no-ad) rules per ITF Beach Tennis Rules: when a game reaches 40-40, one final point is played to decide the game immediately. The receiver chooses which side of the court the server must serve to. This eliminates deuce cycles and keeps games to a maximum of 7 points, making beach tennis faster and more spectator-friendly.

**Ruleset:** ITF
**Rule Reference:** ITF Beach Tennis Rules, Rule 6 (Scoring)

**Related Terms:** no-ad-scoring, tie-break, rally
