# Match - Beach Tennis Glossary

> A match in beach tennis is the complete contest between two players (singles) or two pairs (doubles), typically played as best of 3 sets. Per ITF Beach Tennis Rules Rule 4, the player or pair that wins the majority of sets wins the match. Some tournament formats, particularly consolation rounds, use a single set plus a match tie-break. The match score is tracked as sets won by each side. BeachTennisRef.App tracks serve order, set scores, and match score automatically throughout the full match.

**Ruleset:** ITF
**Rule Reference:** ITF Beach Tennis Rules, Rule 4 (The Match)

**Related Terms:** set, tie-break, no-ad-scoring
