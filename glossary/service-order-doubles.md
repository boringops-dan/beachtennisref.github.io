# Service Order (Doubles) - Beach Tennis Glossary

> In doubles, the service order is the rotation that determines which player serves each game. Per ITF Beach Tennis Rules Rule 8, at the start of each set each team decides which partner will serve first and in which order the players will receive. Within a set, service rotates between the four players in a fixed order: Team A Player 1, Team B Player 1, Team A Player 2, Team B Player 2, then back to Team A Player 1. The order cannot be changed within a set but may be changed at the start of each new set.

**Ruleset:** ITF
**Rule Reference:** ITF Beach Tennis Rules, Rule 8 (Order of Service in Doubles)

**Related Terms:** doubles-formation, tie-break, mixed-doubles
