# Set - Beach Tennis Glossary

> A set is the unit of play consisting of multiple games; the player or team that wins a prescribed number of games first wins the set. In beach tennis, a standard set is won by the first side to win 6 games with a 2-game lead, or 7-6 via a tie-break. Per ITF Beach Tennis Rules, a standard match is best of 3 sets. Some tournament formats use a match tie-break (super tie-break to 10 points) in place of a full third set. The set score is always the most important context for determining serve order and which player or pair serves next.

**Ruleset:** ITF
**Rule Reference:** ITF Beach Tennis Rules, Rule 5 (The Set)

**Related Terms:** tie-break, match, service-order-doubles
